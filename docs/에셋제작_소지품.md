# 소지품 에셋 제작 명세 (인물 명부 · 소지품 탭)

빌더가 그림을 만들 때 보는 단일 기준 문서. 코드/명단 등재는 Claude Code 몫.

## 동작 방식 (요약)
- AI(Gemini)는 물건을 지을 때 **보유 그림 명단에서 제일 비슷한 것 하나**를 골라 딱지를 붙인다 (legacy emoji 선택과 같은 메커니즘).
- 고른 그림은 물건에 박제 — 새로고침해도 안 바뀜.
- 명단에 없는 답 → **공용 문양(폴백)** 으로 자동 처리. 화면은 절대 안 깨진다.
- **그림이 늘수록 선택이 정확해진다.** 전부 다 그릴 필요 없음 — 폴더에 들어오는 대로 한 장씩 살아난다.
- 카테고리는 **제작 가이드**(어디부터 그릴까의 지도)일 뿐, 시스템은 평평한 그림 명단만 안다.

## 공통 사양
- **정사각 512×512, 배경 투명 WebP.** (흰 배경본만 있으면 Claude Code가 배경을 따는 것도 가능 — 투명본이 최선)
- 위치: `public/assets/illust/items/` (파일명은 아래 표 그대로)
- 스타일(확정): **수채+선화의 사실적 소품화** — 밀랍 봉인 편지·코르크 약병·단검 샘플의 화풍이 기준. 차분한 채도(원색 금지), 묶음마다 같은 결 유지.
- 새 그림을 폴더에 넣으면 Claude Code에게 알려줄 것 → 명단(`lib/itemIcons.mjs`)에 이미 47종이 등재돼 있어, **파일명만 표 그대로면 자동으로 떠오른다**(코드 수정 불필요). 표에 없는 새 물건을 그렸을 때만 등재가 필요.
- 그림이 아직 없는 물건은 카드에 **은은한 빈 액자**로 뜬다(화면 안 깨짐). 파일이 들어오는 순간 그 자리에 그림이 채워진다.

## 생성 프롬프트 (앞으로 이걸 꺼내 쓴다)
9장 시트 단위로 뽑는다. **여러 시트를 모을 때 따로 놀지 않게 하는 핵심 = 시점·점유율·대각선·광원을 고정**.
잘 나온 첫 시트를 레퍼런스로 물려 다음 시트를 뽑으면 일관성이 더 좋다.

> 파이어 엠블렘 풍화설월 모티브의 중세 판타지 게임용 아이템 에셋 시트를 제작한다.
> * 1:1 정사각형 캔버스
> * 3×3 그리드, 총 9개 아이템
> * 각 아이템은 하나의 칸에 단독 배치
> * 아이템끼리 겹치지 않으며 충분한 여백 확보
> * 모든 아이템의 크기, 시점, 조명, 채색 밀도, 선화 스타일 통일
> * **시점은 약한 3/4 부감으로 통일한다(정면도·평면도 금지 — '또는'으로 흔들리지 않게 하나로 고정).**
> * **조명은 좌상단에서 오는 부드러운 단일 광원으로 통일한다(하이라이트·음영 방향 동일).**
> * **검·열쇠·깃펜·두루마리처럼 길쭉한 물건은 좌하단→우상단 대각선으로 눕혀 배치한다.**
> * JRPG식 소형 아이템 일러스트
> * 특정 게임 원본 아이템을 복제하지 않고, 중세 판타지 세계관에 어울리는 범용 에셋으로 제작
> * 배경은 누끼따기 쉬운 순백색 #FFFFFF
> * 그림자는 최소화하고 바닥 그림자 금지
> * 프레임, 라벨, 번호, 문자, 장식 테두리 금지
> * 각 아이템은 100px 이하로 축소해도 실루엣이 명확하게 읽히도록 단순하고 선명하게 표현
> * 전체 배색은 앱 주조색 #363645와 어울리는 저채도·중명도 중심
> * 지나치게 실사적이거나 3D 렌더처럼 보이지 않게
> * 한 아이템 안에 여러 물건을 묶지 말고, 단일 오브젝트로 표현
> * 각 아이템은 해당 그리드 칸의 가로·세로 약 65~75%를 점유하도록 배치한다.
> * 실제 크기와 관계없이 모든 아이템의 시각적 크기를 균등하게 맞춘다. 반지, 동전처럼 작은 물건은 확대하고, 검·열쇠처럼 긴 물건은 축소한다.
> * 100px 이하 축소를 고려해 잔무늬, 가는 선, 작은 장식은 줄이고 핵심 실루엣만 선명하게 표현한다.
> * 종교·주술 아이템에는 파이어 엠블렘의 문장이나 기하학적 성표를 사용한다.
> * 모든 아이템은 각 칸의 중앙에 배치하고, 칸 가장자리와 일정한 여백을 유지한다.

마지막 줄에 **그 시트에서 뽑을 9개 물건**을 적어 넣는다. 예: `이번 9개: 서신, 두루마리, 서책, 깃펜, 잉크병, 반지, 펜던트, 머리 장식, 리본`

## 폴백 (최우선 1장)
| 파일 | 그릴 것 |
|---|---|
| `fallback.webp` | 공용 문양 — 어느 물건에 걸려도 어색하지 않은 것(묶인 보자기·매듭 등). **그림이 없는 동안 모든 카드가 이걸 쓴다.** |

## 집계
- 전체 **47장 + 폴백 1장** (끝까지 다 그리면 — 의무 아님)
- **1차 권장 12장(★)**: 서신 · 서책 · 반지 · 양초 · 술병 · 천 조각 · 찻잎 · 꽃 · 단검 · 성표 · 열쇠 · 수상한 꾸러미
- 순서·속도는 자유. 묶음 한 장(예: 3×3)으로 뽑아 낱장으로 잘라도 됨.
